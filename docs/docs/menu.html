Menu

El menú de navegación es importante para un sitio web, ayuda a los usuarios a saltar rápidamente de una sección del sitio a otra. Principalmente, incluye navegación superior y navegación lateral. La navegación superior proporciona todas las categorías y funciones del sitio web. La navegación lateral proporciona la estructura de niveles múltiples del sitio web
