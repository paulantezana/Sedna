Size

Sedna admite un tamaño de botón predeterminado, así como un tamaño grande y pequeño. Si se desea un botón grande o pequeño, establezca la clase de tamaño en grande o pequeño, respectivamente. Omita la propiedad de tamaño para un botón con el tamaño predeterminado.
