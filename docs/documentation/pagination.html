Pagination

A long list can be divided into several pages by Pagination, and only one page will be loaded at a time.

Basic pagination.
<ul class="SnPagination">
    <li class="SnPagination-item disabled"><a href="#" class="SnPagination-link">Previous</a></li>
    <li class="SnPagination-item active"><a href="#" class="SnPagination-link">1</a></li>
    <li class="SnPagination-item "><a href="#" class="SnPagination-link">2</a></li>
    <li class="SnPagination-item "><a href="#" class="SnPagination-link">3</a></li>
    <li class="SnPagination-item "><a href="#" class="SnPagination-link">Next</a></li>
</ul>
